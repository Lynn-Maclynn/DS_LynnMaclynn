# 💎 Prédiction du prix d’un diamant

Dans ce projet, j’ai entraîné un modèle de régression pour estimer le **prix** d’un diamant à partir de ses caractéristiques (carat, couleur, coupe, clarté, etc.), toujours en utilisant la base `diamonds` de Seaborn.

## 🎯 Objectif

- Créer un modèle de régression performant
- Analyser les variables les plus influentes
- Obtenir une prédiction précise du prix

## 📁 Fichiers

- `Predict_price_diamonds.ipynb` : Notebook complet contenant l’analyse, la modélisation et l’évaluation
- Jeu de données chargé via `seaborn.load_dataset("diamonds")`

## 🧰 Technologies

- Python (Pandas, Seaborn, Scikit-learn)
- Régression linéaire, SGD Regressor, GridSearchCV, Ridge_Regressor, Lasso_Regressor
- Jupyter Notebook

## 📊 Résultat

- Modèle final avec **98 % de précision (R²)**
- Visualisations claires des prédictions et de l’erreur

---

💡 Ce projet montre la capacité à construire un pipeline de modélisation, à évaluer les performances et à interpréter les résultats.
